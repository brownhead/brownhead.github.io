Galah

Python 2011 to now

An automated grading system geared towards processing computer programming assignments.

I began this ambitious project Freshman year of college after the System Administrator there, Victor Hill, told me he thought it would be cool if they had something there to automate the grading process. After two years of planning and development I am still working on it regularly. It is currently deployed at the University of California, Riverside, and I hope other schools will start to use it as well.

Superzippy

Python 2013 to now

A new way to distribute multi-file, multi-dependency Python scripts that is just super.

When writing Python scripts, I often run into the unpleasant truth that unless I am creating something for my fellow Python developers, I must restrict my script to a single file. I have tried to laugh in the face of this restriction before and create lengthy installation instructions but the difficulties are always too much to deal with. Superzippy solves this problem by providing an easy way to transform a multi-file, multi-dependency Python script into a single file.

Galah Interact

Python 2013 to now

A Python framework for creating test harnesses to grade students' code.

After a couple weeks of creating test harnesses for Galah by copy and pasting bits of code together, I realised I needed to make creating test harnesses a much simpler process. Thus Chris Manghane (who codeveloped Galah with me) and I created this framework.
